## 关键技术决策
- 无数据库：CSV 即真相，家长可直接修复。
- shared 包统一 Zod schema 与纯逻辑，前后端共享，避免类型漂移。
- 每童一条写队列串行化，跨童并行；requestId 幂等。
- 会话签名 cookie（HMAC，不含 PIN）+ 进程级吊销集合。
- 服务器时区权威，每日上限按本地日历日期。

## 实现中解决的问题
- pnpm 对 `@types/cookie-signature`（每 TS 版本一个变体）和 `@testing-library/dom@^10.5.0`（版本不存在）触发交互式选择/失败：改用本地 `.d.ts` 声明 cookie-signature/cookie-parser，并固定正确版本。
- shared 包 `main` 初版指向 `src/index.ts`，server 生产 `node dist` 无法加载 TS：改为构建到 dist，exports 加 `development` 条件指向 src（Vite 用），default 指向 dist（Node 用）。
- server 相对路径以 packages/server 为 cwd：改用 `fileURLToPath(import.meta.url)` 计算 PROJECT_ROOT，从仓库根解析 config/data/backups/public。
