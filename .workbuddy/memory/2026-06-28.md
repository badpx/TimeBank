## 关键技术决策
- 无数据库：CSV 即真相，家长可直接修复。
- shared 包统一 Zod schema 与纯逻辑，前后端共享，避免类型漂移。
- 每童一条写队列串行化，跨童并行；requestId 幂等。
- 会话签名 cookie（HMAC，不含 PIN）+ 进程级吊销集合。
- 服务器时区权威，每日上限按本地日历日期。

## 鼓励文案弹窗初始位置偏右 bug 修复
- 根因：Confetti 组件用 `left:50% + transform: translateX(-50%)` 居中，但同时套用 `animation: fadeIn`，而 fadeIn keyframes 的 `transform: translateY(8px)→translateY(0)` 覆盖了 translateX(-50%)，导致动画期间 transform 丢失 translateX，元素按 left:50%（左边缘对齐中线）渲染，故初始偏右，动画结束后才回到 transform 静态值居中。
- 修复：新增专用 keyframes `confettiPop`，起止均保留 `translateX(-50%)`（`translateX(-50%) translateY(8px)` → `translateX(-50%) translateY(0)`），并改用该动画。
- 构建通过，服务已重启。
